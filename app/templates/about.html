I created SunnyMinutes in July 2015 as a data science project during the first Healthcare Data Science program organized by Insight Data in Boston, MA.

Tools

Processing: Python 2.7
Storage: mySQL, pysql
Visualization: matplotlib
Web layout: flask, Bootstrap

Source

The entire source code can be downloaded from GitHub.

Buildings

I obtained the geometries of the buildings of Manhattan from OpenStreetMap. Each building (or building part) is represented by an ordered list of nodes (latitude, longitude), and a single hight value.


Sun light

I implemented the astronomical equation to calculate the equatorial coordinates of the sun.

Data aquisition

I called OpenStreetMap's Overpass API with a bounding box containing Manhattan (whose coordinates I read off Google Maps) and filters for "building", and "building:part" tags.

I processed the XML file (size ~100MB) with regular expressions, and extraced the lists of nodes (lat, lon) and height values for each building. I converted the (lat, lon) coordinates to Cartesian coordinates using the assumption that Manhattan is small enough to be taken as flat.

I grouped buildings into 500 m x 500 m blocks.


Processing

From the address I find the geocoordinates using the geocoding package. I determine which block the observer is in, and select all buildings in there and all buildings the neighboring eight blocks too.

From the floor number, I estimate the altitude of the observer assuming three-meter-high floors.

For each building, I determine the (phi, theta) coordinates of their nodes on the obeservers view with trigonometric functions.

I build a skyline view out of the buildings that are visible from the observer's location, and store only this skyline in the memory.

For each week of the year, and each 5 min interval of the day, I determine the sun's position on the observer's sky, and register the times, when the sun is visible, and when it's blocked by the skyline.
